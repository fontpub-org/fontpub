# Fontpub Protocol Assets

This directory contains the executable-spec assets for Fontpub v1.

- `schemas/`
  JSON Schemas for public documents and CLI JSON results.
- `fixtures/`
  Valid and invalid examples used by conformance tests.
- `golden/`
  Canonical JSON byte outputs used for exact serialization tests.

These files are intended to be language-neutral and reusable by multiple implementations.
