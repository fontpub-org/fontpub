# Fontpub

> Identity-native font distribution. The verifiable protocol for open fonts.

## Manifest


Fonts, Reborn as Software.

Historically, fonts have been treated as static files—frozen the moment they are exported. While code has evolved into a dynamic ecosystem supported by version control and CI/CD, fonts have remained isolated in the "single file" era.

Fontpub bridges this gap. We bring the logic and lifecycle of software to the world of open fonts.

### 1. Direct Distribution, No Gatekeepers

Publishing a font should be as free as pushing to a Git repository. Fontpub bypasses central approval processes, empowering creators to transform their repositories into autonomous distribution hubs. We remove the barriers between the source and the desktop.

### 2. Provenance from the Canonical Source

We anchor trust in the repository itself. By leveraging GitHub OIDC, we provide cryptographic proof of provenance—guaranteeing that a font originates directly from its canonical source without tampering. Users no longer rely on central authorities to verify authenticity; they rely on the transparent history of the repository they trust.

### 3. From Static Files to Living Assets

A font is never truly "finished"—it evolves with bug fixes, new glyphs, and refined kerning. Fontpub transforms fonts into versioned, manageable software. Our CLI makes updating your system’s fonts as simple as a package update, ensuring you are always aligned with the latest evolution of the source.
